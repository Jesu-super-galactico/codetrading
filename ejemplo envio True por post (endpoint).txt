ENVIO UN STREAM PARA CAMBIAR LA DIRECCION DE YOUTUBE POR OTRA:

Invoke-RestMethod -Method Post -Body (@{stream_url='https://www.youtube.com/watch?v=ID'} | ConvertTo-Json) -ContentType 'application/json' http://127.0.0.1:8000/set_pending_stream


AQUI HAY TRES EJEMPLOS PARA ENVIAR TRUE COMO ENDPOINT (TIPO POST OBVIO)

1er ejemplo:
$body = @{ apply = $true } | ConvertTo-Json
Invoke-RestMethod -Method Post -Uri http://127.0.0.1:8000/apply_pending_stream -Body $body -ContentType 'application/json'

2do ejemplo:
Invoke-RestMethod -Method Post -Uri http://127.0.0.1:8000/apply_pending_stream -Body ($true | ConvertTo-Json) -ContentType 'application/json'

3er ejemplo:
Invoke-RestMethod -Method Post -Uri http://127.0.0.1:8000/apply_pending_stream -Body 'true' -ContentType 'application/json'
